# Chatrel Report Knowledge

Public, redacted reference knowledge for `chatrel-gap-review`.

This package contains distilled relationship-analysis dimensions, report audit coverage rules, output fields, and literature references. It excludes raw chat logs, generated run logs, private report filenames, local absolute paths, PDF files, and full evidence bundles.

## Contents

- `latest_gap_audit.md`: redacted coverage snapshot for the current report-audit schema.
- `latest_distilled_knowledge.md`: merged redacted dimension knowledge.
- `distilled/dimensions/*.md`: one file per relationship-analysis dimension.

## Privacy Scope

The published files are designed as reusable analysis knowledge. They keep reusable metrics and rules, while removing private machine paths and chat-report identifiers.
