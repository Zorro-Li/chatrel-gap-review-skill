# References Package Map

This skill ships three public reference packages:

- `chatrel_report_knowledge/`: report-gap audit snapshots and distilled dimension knowledge.
- `foundation_knowledge/`: the public foundation layer for dyadic relationship knowledge, including taxonomy and sanitized paper manifests.
- `chat_analysis_pipeline/`: the method layer from chat-record analysis to relationship report generation.

Public packages keep reusable rules, schemas, prompts, manifests, and workflow documents.

Local-only assets stay outside this repository:

- raw chat records
- private report files
- SQLite databases
- retrieval chunk stores
- PDFs
- generated evidence bundles
- local run logs
