# 视频逐字稿工作区

这个文件夹把“基于微信聊天记录分析两个人关系”整理成可拍短视频的逐字稿。

## 系列定位

主题：从聊天记录里提取关系信号。

表达目标：

- 让观众理解聊天记录里能观察到哪些关系线索。
- 把每个线索讲成一个清楚的视频主题。
- 每条视频都能回到论文知识库里的证据。
- 口播结尾给出一个具体判断方法。

## 使用方式

每篇脚本分四块：

- `逐字稿`：直接念。
- `拍摄节奏`：剪辑节拍。
- `画面提示`：配屏幕录制、字幕或图表。
- `知识库检索词`：回查论文证据。

## 默认口播风格

- 直接讲结论。
- 每条视频只讲一个核心特征。
- 用微信聊天记录做例子时，展示结构和指标，隐去真实内容。
- 所有判断都表达为概率信号。
- 结尾给出一个可执行的观察动作。
