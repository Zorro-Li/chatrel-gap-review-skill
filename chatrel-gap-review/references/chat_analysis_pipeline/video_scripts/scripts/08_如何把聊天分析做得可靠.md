# 08 如何把聊天分析做得可靠

建议时长：60-90 秒

## 逐字稿

基于聊天记录做关系分析，最重要的是可靠。

可靠的意思是：每一个判断，都要能回到可观察证据。

你说两个人关系强，就要拿出频率、互惠、回复速度和话题延续。

你说两个人有信任，就要拿出自我披露、及时回应、稳定支持和敏感话题的承接。

你说一个人更主导，就要拿出发起次数、指令比例、话题控制和拒绝空间。

你说关系出现修复，就要拿出冲突后的重新开启、道歉、解释和互动恢复。

聊天分析要分三层输出。

第一层是事实：谁在什么时候发了什么。

第二层是特征：回复速度、互动频率、情绪支持、语言对齐、冲突修复。

第三层是解释：这些特征共同指向什么关系状态。

最容易出错的地方，是跳过第二层，直接从一句话判断一个人。

可靠的做法，是先统计，再分段，再解释。

比如，不说“这个人冷淡”。你可以说：在最近三周里，TA 的平均回复间隔变长，主动发起减少，情绪支持回应下降，所以关系投入可能在降低。

这才是可检查的分析。

聊天记录给我们的不是读心术，而是一套互动证据。

把证据整理清楚，关系判断才有价值。

## 拍摄节奏

- 0-10 秒：定义可靠。
- 10-45 秒：用四类判断举例。
- 45-70 秒：讲事实、特征、解释三层。
- 70-90 秒：给出规范表达方式。
