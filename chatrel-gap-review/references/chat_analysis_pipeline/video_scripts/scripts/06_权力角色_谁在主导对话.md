# 06 权力角色：谁在主导对话

建议时长：60-90 秒

## 逐字稿

聊天记录里，权力关系经常藏在很小的句子里。

谁经常提要求，谁经常解释。

谁经常安排时间，谁经常配合时间。

谁经常发判断，谁经常发疑问。

谁经常结束话题，谁经常试图把话题拉回来。

这些都是角色线索。

权力不一定表现为强硬。很多时候，它表现为默认规则。

比如，一个人说“你现在过来一下”，另一个人立刻解释自己为什么来不了。这里的重点不是这句话多强，而是这种模式是否反复出现。

做聊天分析时，可以看四个指标。

第一，指令比例。谁更常用“你去”“你要”“你帮我”。

第二，解释比例。谁更常解释自己的行为。

第三，提问比例。谁更常通过提问推动对话。

第四，话题控制。谁更常开启、切换和结束话题。

如果两个人都能发起、都能拒绝、都能解释、都能改变话题，这段关系的角色更平衡。

如果长期只有一个人安排，另一个人配合，关系里就有明显的主导结构。

判断权力关系的关键，是看长期重复的互动模式。

一两句强势的话说明不了什么。一个稳定重复的对话结构，才是关系里的角色分配。

## 拍摄节奏

- 0-10 秒：讲权力藏在小句子里。
- 10-40 秒：列出角色线索。
- 40-70 秒：讲四个指标。
- 70-90 秒：收束到长期模式。
