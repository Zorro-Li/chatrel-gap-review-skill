# 03 关系强度：频率、互惠和回复速度

建议时长：60-90 秒

## 逐字稿

判断两个人关系强不强，先看三个指标：频率、互惠、回复速度。

频率，就是两个人联系得多不多。

互惠，就是这段联系是不是两个人都在投入。

回复速度，就是对方在这段关系里的优先级。

只看频率会误判。一个人一直发，另一个人很少回，这叫高频单向投入。两个人都发，话题能来回接，这才接近高互动关系。

所以，第一步看消息数量。A 发了多少，B 发了多少。

第二步看发起次数。谁更常开启一段聊天。

第三步看回复间隔。谁总是秒回，谁总是隔很久。

第四步看互惠程度。比如 A 发十句，B 只回一个“嗯”，这在数量上有回应，在互动质量上很弱。

真正有价值的是组合判断。

高频、高互惠、低延迟，通常代表关系连接强。

高频、低互惠、高延迟，通常代表关系投入不平衡。

低频、高质量、稳定回复，可能代表关系节奏慢，但关系仍然稳定。

做聊天记录分析时，你可以先画一张两人互动热力图：横轴是时间，纵轴是两个人，颜色代表消息密度。

这张图一出来，关系强度的变化会很直观。

你会看到谁在靠近，谁在后退，谁在维持连接。
