# 05 共情支持：聊天里的安全感

建议时长：60-90 秒

## 逐字稿

聊天记录里能看出一个人有没有给对方安全感。

安全感不是一句“我在”，而是一套稳定的回应方式。

当对方表达压力时，你是忽略，转移话题，给建议，还是先接住情绪？

当对方表达脆弱时，你是评价对错，还是表达理解？

当对方遇到困难时，你是只说“加油”，还是提供具体帮助？

这些都属于支持性沟通。

在聊天记录里，可以把支持拆成四类。

第一类是情绪确认。比如“这件事确实很难受”。

第二类是理解和共情。比如“我能理解你为什么会这样想”。

第三类是实际帮助。比如“我帮你一起看一下”。

第四类是陪伴承诺。比如“你先说，我听着”。

关系里真正稳定的支持，通常有两个特点。

第一，它来得及时。对方需要回应的时候，你能出现。

第二，它能对准需求。对方需要被理解时，你给理解；对方需要解决问题时，你给方案。

所以，分析聊天记录时，不要只数“安慰的话”有多少。

更重要的是看：支持有没有出现在关键时刻，支持有没有被对方接住。

如果一个人经常表达压力，另一个人经常能接住、回应、延续话题，这段关系的情绪承载能力通常更强。

## 拍摄节奏

- 0-10 秒：讲安全感来自回应方式。
- 10-40 秒：举压力、脆弱、困难场景。
- 40-65 秒：拆四类支持。
- 65-90 秒：给出判断标准。
