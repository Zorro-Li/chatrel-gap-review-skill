# 07 冲突、破裂和修复

建议时长：60-90 秒

## 逐字稿

聊天记录里最值得看的，往往不是冲突本身，而是冲突之后发生了什么。

两个人吵架很常见。关系差异真正拉开的地方，是破裂之后有没有修复。

一段冲突通常有三个阶段。

第一阶段是升级。

比如语气变硬，回复变短，否定词变多，解释变密集。

第二阶段是破裂。

比如沉默、转移话题、停止回应、退出对话。

第三阶段是修复。

比如有人重新开启话题，有人道歉，有人解释感受，有人提出下一步怎么做。

分析聊天记录时，你可以把冲突窗口单独切出来。

不要只看谁说了狠话。要看谁先试图修复，谁接住了修复，修复之后互动有没有恢复。

有些关系冲突很少，但一冲突就长时间冷掉。

有些关系冲突不少，但每次都能回到沟通。

后者的关系韧性通常更强。

所以，一个实用指标是“修复速度”。

从冲突高点到重新正常对话，中间隔了多久？

另一个指标是“修复质量”。

修复只是换个话题，还是双方真的回应了刚才的问题？

真正值得记录的，是这段关系处理压力的方式。

关系的稳定性，常常体现在冲突之后。

## 拍摄节奏

- 0-8 秒：冲突之后更值得看。
- 8-35 秒：讲升级、破裂、修复。
- 35-65 秒：讲修复速度和修复质量。
- 65-90 秒：收束到关系韧性。
